Integrate ParkFlow parking management booking widget into your WordPress site.

== Description ==
This plugin allows you to easily integrate the ParkFlow booking widget into your WordPress site.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/parkflow-widget`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure the plugin settings under "ParkFlow" in the admin menu
4. Use the shortcode [parkflow] to display the booking widget
